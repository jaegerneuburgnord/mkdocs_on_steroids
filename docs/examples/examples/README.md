# Example Archives

This directory contains downloadable example projects.

## Available Examples

- **http_client.zip** - HTTP client implementation example
- **thread_pool.zip** - Thread pool implementation example
- **parallel_algos.zip** - Parallel algorithms example
- **all_examples.zip** - All examples combined

## Usage

Download and extract the ZIP files to explore complete, working examples of various library features.

Each example includes:
- Source code
- CMakeLists.txt for building
- README with instructions
- Sample data (where applicable)
