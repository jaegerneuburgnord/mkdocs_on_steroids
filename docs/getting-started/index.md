# Getting Started

Willkommen beim Getting Started Guide für die C++ Advanced Library! Dieser Abschnitt führt dich durch die Installation, Konfiguration und erste Schritte mit der Bibliothek.

## Was du lernen wirst

In diesem Abschnitt behandeln wir:

- :fontawesome-solid-download: **Installation** - Wie du die Bibliothek installierst und einrichtest
- :fontawesome-solid-rocket: **Quick Start** - Erstelle dein erstes Programm in 5 Minuten
- :fontawesome-solid-hammer: **Build System** - Integration in CMake, Make und andere Build-Systeme
- :fontawesome-solid-puzzle-piece: **Dependencies** - Verwaltung von Abhängigkeiten

## Voraussetzungen

Bevor du beginnst, stelle sicher, dass du Folgendes hast:

### Compiler

!!! requirement "Mindestanforderungen"
    - **GCC** 10.0 oder höher
    - **Clang** 11.0 oder höher
    - **MSVC** 19.28 (Visual Studio 2019 16.8) oder höher

### Build Tools

- **CMake** 3.16 oder höher
- **Git** für das Klonen des Repositories

### Optional

- **Conan** oder **vcpkg** für Dependency Management
- **Ninja** für schnellere Builds
- **Doxygen** für das Generieren von API-Dokumentation
